# Software Defect Prediction

This project uses Machine Learning techniques to predict software defects in software modules.

## Dataset

The datasets used in this project are obtained from the NASA PROMISE Repository.

Download them from:
http://promise.site.uottawa.ca/SERepository/datasets-page.html

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Machine Learning

## Features
- Data preprocessing
- Feature selection
- Model training
- Defect prediction
